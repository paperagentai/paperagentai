Configuration

Set your research question, domain context, and API settings. Then run the pipeline backend locally or on a server.

🔌 Backend Connection

Where the Flask backend is running. Default: http://localhost:5000. Change this if your backend is on a different host.

🔑 LLM API Settings

Your LLM API key. Sent to the backend — never stored by GitHub Pages.

🎯 Research Question

One sentence — the central question. This anchors every pipeline step.

📚 Research Context

Domain-specific details: venue, literature domains, frameworks, experiment constraints, evaluation approach, tools.

📖 Recent Papers

Post-training-cutoff papers. Format:
## Paper N / **Title:** ... / **URL:** ...

📝 Feedback (for Step 5f)

Optional reviewer feedback — injected into the review step.

Run Pipeline

Execute steps individually or run the full pipeline end-to-end. Requires the backend to be running.

📤 Output 

Select a step and click Run, or click "Run All Steps" to begin.

Log Files

View and manage generated outputs from each step. Logs are stored on the backend.
