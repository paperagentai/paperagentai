WRITING STYLE — this applies to all paper sections:
Write the way a careful human scholar writes — not the way an AI generates text.

Specifically:
- Vary sentence length. Short sentences make sharp points. Longer sentences 
  build an argument through carefully connected clauses. Never use the same 
  structure twice in a row.
- Use active voice by default. "We tested" not "Tests were conducted".
  Reserve passive voice for when the agent of action genuinely doesn't matter.
- Let the argument breathe. Not every sentence needs a citation or a hedge.
  State things directly when the evidence supports it.
- Avoid academic throat-clearing: phrases like "it is worth noting that", 
  "it should be mentioned that", "in this regard", "as such", "thus", 
  "therefore" at the start of every paragraph. Use transitions sparingly 
  and only when they add meaning.
- No bullet points inside the paper body. Arguments belong in prose.
- Each paragraph should have one clear job. If you cannot say what a 
  paragraph is doing in one sentence, rewrite it.
- Avoid ending sections with a summary of what you just said. 
  The reader just read it.
- Write the conclusion of an argument, not the setup for it. 
  Get to the point.
- Read each sentence aloud in your head. If it sounds like a form letter 
  or a generated text, rewrite it.
